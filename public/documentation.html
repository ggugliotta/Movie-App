Allow new users to register /users POST A JSON object holding data about the new user containing an ID, name, and favorite movie place holder. Example: { id: 1, name: user1, favoriteMovie: [] } A Status Code (201) indicating that the registration was successful.
Allow users to update their user info (username) /users/:id PUT A JSON object holding data about the new user containing an ID, name, and favorite movie place holder. Example: { id: 1, name: user1, favoriteMovie: [] } A Status Code (200) indicating that the registration was successful.
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) /users/id/:movieTitle POST A JSON object holding data about the new user containing an ID, name, and favorite movie place holder. Example: { id: 1, name: user1, favoriteMovie: [] } A status code (200) sent to the console that lets you know that the movie title has been added to the user's array.
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) /users/id/:movieTitle DELETE A JSON object holding data about the new user containing an ID, name, and favorite movie place holder. Example: { id: 1, name: user1, favoriteMovie: [] } A status code (200) sent to the console that says that the movie title has been deleted from the user ID's array.
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later). /users/:id DELETE A JSON object holding data about the new user containing an ID, name, and favorite movie place holder. Example: { id: 1, name: user1, favoriteMovie: [] } A status code (200) sent to the console that says that the user ID has been deleted or an error code that says no such user.
