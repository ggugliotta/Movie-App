Allow new users to register /Users/:name POST A JSON object holding data about the new user containing an ID, name, and favorite movie place holder. Example: { "Name": "Roman Flores", "Username": "romiethehomie", "Password": "Ilovecats2023!", "Email": "romanflores@gmail.com", "Birthday": "1995-01-06" } A Status Code (201) indicating that the registration was successful.
Allow users to update their user info (username) /users/:Username PUT A JSON object holding data about the new user containing an ID, name, and favorite movie place holder. Example: { "FavoriteMovies": ["64fb5845e6a86f94b742a245"], "_id": "64fb68fae6a86f94b742a24a", "Name": "Gabriella Gugliotta", "Username": "gabriella_gugliotta", "Password": "Codingisfun2023", "Email": "gabriella.g@gmail.com", "Birthday": "1996-03-31T00:00:00.000Z" } A Status Code (200) indicating that the registration was successful.
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) /users/:Username/movies/:MOVIEID POST A JSON object with user updated information. Example: "FavoriteMovies": ["64fb5845e6a86f94b742a245"], A status code (200) sent to the console that lets you know that the movie title has been added to the user's array.
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) /movies/:Title DELETE A notification that says "Title was deleted" Example: "Selena was deleted." A status code (200) sent to the console that says that the movie title has been deleted from the user ID's array.
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later). /users/:id DELETE A notification "username was deleted". Example: "Ilovemovies was deleted." A status code (200) sent to the console that says that the user ID has been deleted or an error code that says no such user.
